# ITAI 4370 Course Portfolio — GitHub Pages Package

This folder is a complete static website prepared for Elizabeth Carrillo's ITAI 4370 Course Portfolio. It includes selected works, three reflective essays, detailed project documentation, assessment artifacts, growth evidence, references, code, figures, and downloadable copies of the Final Exam Portfolio.

## Publish it with GitHub Pages

1. Sign in to GitHub and create a **new public repository** named `itai-4370-course-portfolio`.
2. Open the new repository and choose **Add file → Upload files**.
3. Upload **everything inside this folder**, including `index.html`, `styles.css`, `script.js`, the `assets` folder, and `rubric-checklist.md`. Do not upload only the ZIP.
4. Commit the files to the `main` branch.
5. Open **Settings → Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select the `main` branch and the `/ (root)` folder, then save.
8. Your link will follow this format:
   `https://YOUR-GITHUB-USERNAME.github.io/itai-4370-course-portfolio/`
9. Open the link in a private/incognito window and test the navigation, images, code downloads, PDF, and DOCX.
10. Submit that live link in Canvas before July 20, 2026 at 11:59 p.m.
